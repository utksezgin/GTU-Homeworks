-Build and run to start the game.
-Choose a game mode, Single Player to play against AI, Multi Player to play against a human.
-Blue is player 1, Red is player 2, Winning Cells are highlighted by yellow color.
-Enter a number greater than 4 for board size and start playing the game.

Connect Four:
Try to get 4 cells in a row and prevent your opponentvertical, horizantal, diagonal and prevent your opponent to win the game.
	
Known Bugs:
Cannot change game mode after initially pick it. Must restart the game in order to change the game mode.
